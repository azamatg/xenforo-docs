{{% bq %}}Добавляет HTML для отображения тега BUTTON (кнопка) с нужными классами и иконкой{{% /bq %}}

Простой пример использования:

```html
<xf:button icon="save"></xf:button>
```

## Атрибуты

{{% table_class %}}
| Атрибут | Описание |
| --- | --- |
| `icon` | Класс иконки для отображения на кнопке |
{{% /table_class %}}
