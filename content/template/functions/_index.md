## Вызов функции

Вызов функции в шаблоне:

```html
{{ phrase('new') }}
```

## Простой список функций с краткими описаниями

- `phrase()`
- `link()`
- `number()`
- `date_time()`
- `bb_code()`
- `page_title()`
